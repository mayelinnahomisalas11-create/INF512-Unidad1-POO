# INF512 - Unidad 1 - Programación Orientada a Objetos
Este repositorio corresponde a la evidencia práctica de la Unidad 1
de la asignatura INF512.

El proyecto contiene un ejemplo básico de Programación Orientada a Objetos
implementado en C#, donde se aplican los conceptos de:
- Clase
- Atributos
- Métodos
- Encapsulamiento

El objetivo es demostrar el uso correcto de la POO mediante un ejemplo sencillo.
